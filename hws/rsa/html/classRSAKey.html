Detailed Description

Holds a RSA key, either public or private (or possibly both). In addition to n, which must be in all keys, one (or both) of d and e* must be set. Which one(s) will determine whether it's a public key (if only e is set), a private key (if only d is set), or a dual key (if both d and e are set). This class is meant to hold the values, and has no methods itself.
